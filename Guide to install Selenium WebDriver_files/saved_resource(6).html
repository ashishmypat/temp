                                                                                    Lorri Justyna J • a year ago

                                                                                    Justyna, After clicking the "Add External Jar Files..." button, Windows Explorer should open. From here, navigate to the folder in which you installed the Selenium Java Driver back in Step 3. For me, since I installed a newer version of the driver, the folder was named this: C:\selenium-2.46.0. From within this folder you will see 2 *.jar files. Select both (using the Ctrl key on your keyboard) and then click "Open" which will add them to the Java Build Path in the Eclipse Window. You're not finished quite yet though. After you do this, click "Add External Jar Files..." again and in Windows Explorer, go here: C:\selenium-2.46.0\libs. From here, select all of your *.jar files and click "Open" to add these files to your Java Build Path as well. (Think of it as just copying all of the .jar files located in your c:/selenium-2.46.0 and c:/selenium-2.46.0/libs folder into Eclipse so you can use them later). Then click "OK" in Eclipse. I hope this helps!
